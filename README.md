# CRM Trucking Insurance

Web CRM system for a trucking insurance agency in the United States.

## Purpose

To centralize commercial operations, client management, policy tracking, and administrative control in a modern, secure, and scalable system.

## Technologies

- Backend: Flask (Python)
- Frontend: React
- Database: SQLite / PostgreSQL
- Authentication: JWT

## Architecture

This project is developed following a professional phase-based architecture with strict version control and documented milestones.

Under active development.
